Module: about

The about module serves to encapsulate the class of the same name. The About class extends the default Command class and handles requests for information about a given user passed as a parameter in the command invocation. The class queries the MediaWiki API for information about the given user, including gender, registration date, date of last edit, user group membership, etc.
