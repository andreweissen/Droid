Module: verify

The verify module serves to encapsulate the class of the same name. The Verify class, arguably the most important class after Client, handles the verification of users seeking access to the server by interfacing with the MediaWiki and Fandom APIs to determine if the user is a contributor to the wiki. The need for this functionality was ultimately responsible for the application's creation in the first place, and the class was among the first written and tested.
